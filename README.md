# SystemAuth

Sistema de autenticação JWT desenvolvido com FastAPI, Python, MySQL, HTML, CSS e JavaScript.

## Funcionalidades

- Login de usuários
- Cadastro de usuários
- Autenticação JWT
- Rotas protegidas
- Integração com MySQL
- Interface responsiva

## Tecnologias

- Python
- FastAPI
- MySQL
- HTML
- CSS
- JavaScript
